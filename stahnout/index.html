Co dál

  1. Spusťte stažený soubor Zapisovacka-Setup.exe a projděte instalací.
  2. Windows SmartScreen může zobrazit varování „Neznámá aplikace" — beta instalátor zatím není podepsaný certifikátem vydavatele. Klikněte na Další informace → Přesto spustit.
  3. Zapisovačka se po instalaci usadí v liště. Při prvním spuštění si stáhne model pro přepis a pak už nahrává a přepisuje sama.

In English: run the downloaded Zapisovacka-Setup.exe; if Windows SmartScreen shows a warning, click "More info → Run anyway". The app's interface is currently in Czech.
